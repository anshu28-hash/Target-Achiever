# Target Achiever 🎯

**Target Achiever** is an intelligent goal-planner web application designed to help users decompose high-level objectives into tactical, macro execution pipelines. It features a sleek, dark user interface, dynamic timeline synchronization, and voice interactions to provide an immersive tracking experience.

## ✨ Features
* **Goal Decomposition:** Breaks down custom goals into actionable timelines based on defined strategy metrics.
* **Context Engine:** Provides intelligent insights and progress context dynamically.
* **Voice Capabilities:** Features voice responses powered by ElevenLabs for an engaging workflow.
* **Pipeline Archive:** Safely logs completed goals and tracks overall progress validation.

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **AI Integration:** Google Gemini API
* **Voice Integration:** ElevenLabs API
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)

## 🚀 Getting Started Locally

### Prerequisites
* Python 3.x
* A Gemini API Key
* An ElevenLabs API Key

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/anshu28-hash/Target-Achiever.git](https://github.com/anshu28-hash/Target-Achiever.git)
   cd Target-Achiever






1.   Install dependencies:

pip install -r requirements.txt

2.  Configure Environment Variables:

Create a .env file in the root directory and add your secret API keys:
Code snippet

GEMINI_API_KEY=your_gemini_api_key_here
ELEVENLABS_API_KEY=your_elevenlabs_api_key_here

Run the Application:
python app.py
